The "Hyper-Tactile OS" Expansion Prompt

"Act as a Senior Creative Technologist. I am building a portfolio site called esnupi, a React/TS app mirroring a late-90s Mac OS 8 desktop over a live Hydra-synth wallpaper. My goal is to transform this from a static recreation into a living, 'haunted' digital artifact. Provide a technical and design roadmap for the following features:
1. Tactile Depth & 'Physical' UI

    Dynamic Bezels: Suggest a CSS/Tailwind strategy for 'Resolution-Aware Shadowing.' As windows move toward the edges of the screen, their drop shadows should shift based on a central 'light source' (the cursor or center of the screen) to simulate depth.

    The 'Felt' Physics: Since my icons are felt illustrations, how can I use GSAP to add a 'jiggle' or 'squish' physics when they are dragged and dropped, making them feel like physical fabric patches?

    Audio-Visual Feedback: Create a 'Mechanical Hard Drive' sound profile. When an app is 'loading' (even if instant), how can I play a randomized sequence of low-volume 'disk seek' chirps synchronized with a subtle 1px screen jitter?

2. Expanded 'Working' Apps

    The 'System Profiler' (Deep Portfolio): Instead of a standard project list, design a 'System Profiler' app. It should categorize projects as 'Drives' or 'Kernel Extensions.' Selecting one shows a hex-dump aesthetic that fades into a clean, modern case study.

    Control Panel (Shader Lab): Create a 'Appearance' Control Panel that allows the user to live-tweak the Hydra-synth parameters (speed, noise, kaleid) via sliders, essentially letting the user 'overclock' the site's background.

    Scrapbook (Persistent Snippets): A Drag-and-Drop 'Scrapbook' app where users can drag text from the Terminal or images from the Browser into a persistent local-storage list of 'clippings.'

3. Y2K Easter Eggs & 'Ghost' Events

    The 'Midnight Glitch': Beyond the 'Sleep Nag' clock, describe a rare event (1% chance) where the desktop icons begin to drift slowly toward the center of the screen as if by gravity after 2 minutes of inactivity.

    Terminal 'Secret' Paths: Add a hidden /dev/mind directory in the xterm.js filesystem. If the user cats a specific file, it should trigger a 10-second 'Memory Leak' animation—windows duplicating and cascading across the screen before 'self-healing.'

    Desktop 'Inhabitants': Conceptualize a 'Desktop Pet' (like e-Sheep or BonziBuddy, but stylized in 8-bit felt) that occasionally walks across the menu bar and reacts to the user's open windows.

4. Immersion & Polish

    Boot Sequence 'Vapor': Enhance the BootSequence. How can I add a 'Warm-up' filter that starts the screen in blurry grayscale and slowly focuses and gains color saturation over 5 seconds to simulate a CRT tube warming up?

    Contextual Cursors: Provide a list of custom cursor states (the 'Watch' wait icon, the 'Finger' for links, and a 'Magnifying Glass' for the Photobook) that match the System 8 pixel-art style perfectly.

5. Technical Implementation

    State Management: Suggest a way to use Zustand or Context to ensure that if a user 'Crashes' a window (via a 'Bomb' icon easter egg), the 'System' logs the error in the Terminal's syslog in real-time.

    Performance: How should I throttle the p5 grain and Hydra shaders when multiple windows are open to maintain 60fps on M2/M4 Mac hardware?"

Why this works for esnupi:

    The 7-Click Rule: You already have the 7-click Apple menu egg; this prompt expands that "curiosity-first" philosophy.

    Felt Consistency: It leans into your unique "felt" aesthetic to make the UI feel cohesive rather than just a generic Mac clone.

    Technical Synergy: It utilizes your existing GSAP, Hydra, and xterm.js setup to create "cross-talk" between apps.
