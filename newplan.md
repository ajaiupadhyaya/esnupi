Redesign this portfolio to feel like a physically assembled scrapbook interpreted through a minimalist / brutalist digital lens. The current design looks generically “vibecoded” — too clean, too default, and lacking intention. Fix that.

Core Design Direction
Treat the page like a flatbed scan of a scrapbook, not a modern website
Everything should feel placed by hand, not auto-aligned
Embrace intentional imperfection:
Slight misalignments
Uneven spacing
Cropping that feels human, not algorithmic
Brutalism = raw, honest, functional, not empty whitespace
Layout System
Break away from standard grids
Use a loose, collage-like structure:
Overlapping elements
Occasional rotation (±1–3 degrees max, subtle)
Mixed sizing (some images large and dominant, others small like scraps)
Maintain visual balance, but avoid symmetry
Typography
Replace default fonts with:
One neutral grotesk (e.g., Helvetica/Arial style)
One typewriter or serif accent
Use typography as material:
Vary sizes aggressively
Occasional ALL CAPS labels
Small annotation-style text (like notes in a scrapbook)
Avoid perfect hierarchy—make it feel discovered
Visual Texture (IMPORTANT)
Introduce paper-like qualities:
Off-white background (not pure white)
Subtle grain/noise
Light shadowing as if elements are slightly raised
Add scanned artifacts:
Tape strips
Underlines
Hand-drawn marks
Keep it subtle—no gimmicks
Components
Images should feel like:
Printed photos placed on a page
Slight borders or inconsistent framing
Sections should feel like:
Pages or clusters, not clean div blocks
Hover interactions:
Slight lift, tilt, or shadow (like picking something up)
No flashy animations
Color
Mostly black, off-white, and grayscale
Very limited accent color (used sparingly, like a pen mark)
What to REMOVE
Perfect centering everywhere
Even padding/margins
Generic card UI
Smooth, overly polished animations
Anything that looks like a template
Final Goal

The site should feel like:

A MoMA-style experimental portfolio
A personal archive
A physical object scanned into digital form

When someone sees it, they should think:
“This was designed carefully” — not “this was generated.”
